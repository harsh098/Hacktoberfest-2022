# Program 

## Question
> WAP To Print First 10 Even Number ?

## Solution
>  #include `<iostream>`
>  
>  using namespace std;
>   
>  int main()
>  {
>  
>  cout<<"First 10 Even Numbers are : ";
>  
>  for(int i=1; i<=10; i++)
>  {
>     
>     int n = 2*i;
>     
>     cout<< n <<", ";
>     
>     return 0;
>  }
